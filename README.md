# Vox Biblios 

## a personal text-to-podcast generator

Vox Biblios is a personal podcast generator built using Python and the `feedgen` library. It allows users to convert text files into podcast episodes and publish them to an RSS feed. 
### Features

1. **Text-to-Podcast Conversion**: Converts text files into podcast episodes.
2. **RSS Feed Generation**: Generates an RSS feed for the podcast.
3. **Polly Integration**: Utilizes Amazon Polly to convert text to speech and upload the audio to an S3 bucket.
